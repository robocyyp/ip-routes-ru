# ip-routes-ru
Маршруты для роутеров Keenetic/Netcraze для пропуска траффика сайтов, которые работают только в РФ

Сценарий использования:

- У вас на роутере Keenetic/Netcraze поднят какой-то VPN, через который вы пропускаете весь свой трафик и раздаете на ноутбуки, телефоны, телевизор, приставки и т.д.
- У вас отлично работают зарубежные сайты, но некоторые российские сайты перестают работать, вы не хотите постоянно переключаться между сеткой с впн и без впн.
- Вы добавляете маршруты из файла routes.bat для вашего основного Интернет-подключения, чтобы сайты, которым требуется работа из РФ проходили через ваш "обычный" интернет.

На данный момент добавлены маршруты для основных известных площадок стриминга фильмов, гос.сайтов, бюджетных организаций, некоторых банков. На текущий момент для моей повседневной деятельности этого хватает, если найдете что не работает - создавайте issues, дополним.

Как добавить маршруты:

0. все это имеет смысл, только если у вас уже настроен на роутере впн и он работает.
<img width="1133" height="1269" alt="image" src="https://github.com/user-attachments/assets/a9df2ac6-9cb0-45c7-bed5-4edf32f885e6" />

1. Переходим в ваш ЛК роутера на страницу маршрутов для IPv4: https://192.168.1.1/staticRoutes/ipv4   (Адрес ЛК может отличаться, предполагается, что вы знаете адрес вашего ЛК, если уже настроили впн)

2. Нажимаем Импорт, в интерфейсе выбираем ваше подключение, через которое осуществляется подключение к вашему интернет-провайдеру, чтобы через него пропускать трафик в обход впн

3. Добавляем файл routes.bat, нажимаем загрузить

4. Готово, теперь у вас должны работать и заблокированные сайты и сайты РФ на всех устройствах, к которым подключен роутер.
